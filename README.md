# easyjob

#### 介绍

基于uni-app框架的考研政治题目复习系统(毕设)

#### 项目技术:

前端:uni-app+Vue3+Element Ul

后端:Springboot+Mybatis+mysql+redis

#### 项目描述

设计并开发了一款考研政治智能复习系统，包含移动端用户复习系统与Web端后台管理系统系统融合个性化推荐算法与知识点掌握模型，实现智能组卷、错题分析、学习统计等功能。

#### 项目功能

##### 移动端:

①实现用户注册登录、个人信息管理、题目练习(背诵/阅读模式)、在线考试(计时、答题卡、异常恢复)、错题本、收藏夹等功能。

②)开发学习统计模块，集成签到、数据可视化(近5次考试正确率、知识点掌握度)，并构建用户-知识点掌握矩阵，基于皮尔逊相关系数计算用户相似度实现个性化题目推荐。

##### 后台管理端:

①实现题库管理(分类管理、题目增删改查、Excel批量导入、OCR识别录入)、用户管理、轮播图管理、版本更新管理(灰度发布/全网发布)、问题反馈处理等功能。

②)开发数据报表模块，使用Echarts展示系统关键数据(用户数、题目量、反馈量等)

③实现角色权限管理(菜单管理、权限分配)，基于AOP实现登录校验与权限控制。
